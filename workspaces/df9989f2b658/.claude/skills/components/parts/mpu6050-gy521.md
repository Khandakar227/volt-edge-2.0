# GY-521 (MPU6050) — 6-DoF IMU breakout

**Aliases:** MPU6050, MPU-6050, GY521, GY-521, 6-axis IMU, accelerometer+gyro breakout
**Category:** sensor breakout board (I²C)

## Physical

- **Board size:** ≈ 20 × 16 mm (variants report ~21.2 × 15.6 mm).
- **Connector:** a **single inline row of 8 pins**, 2.54 mm pitch (SIP header).
  It is **not** a DIP-8 IC — do not use a `dip8` footprint.

## Pinout (single 8-pin header, in order)

| Pin | Label | Function |
|----|-------|----------|
| 1 | VCC  | 3.3–5 V supply (onboard regulator) |
| 2 | GND  | ground |
| 3 | SCL  | I²C clock |
| 4 | SDA  | I²C data |
| 5 | XDA  | auxiliary I²C data (aux master) |
| 6 | XCL  | auxiliary I²C clock |
| 7 | AD0  | I²C address select (low = 0x68, high = 0x69) |
| 8 | INT  | interrupt output |

## tscircuit modeling

```tsx
<pinheader
  name="MPU6050"
  pinCount={8}
  pitch="2.54mm"
  footprint="pinrow8"
  gender="male"
  schFacingDirection="left"
  showSilkscreenPinLabels={true}
  pinLabels={["VCC", "GND", "SCL", "SDA", "XDA", "XCL", "AD0", "INT"]}
/>
```

Reference the pins in traces by label, e.g. `.MPU6050 > .SCL`, `.MPU6050 > .SDA`.
Only the first four pins (VCC/GND/SCL/SDA) are needed for a basic I²C hookup;
leave XDA/XCL/AD0/INT unconnected unless required.
